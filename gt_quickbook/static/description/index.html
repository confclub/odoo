Odoo online Quickbooks (bi-directional) connector

Odoo quickbooks connector will help you sync all the quickbooks data in odoo which is bi-directional

This module will provide with below features

  • Import / Export Account
  • Import / Export Taxes
  • Import / Export Payment method
  • Import Payment terms
  • Import Department
  • Import / Export Customers
  • Import / Export Vendors
  • Import / Export Employees
  • Import / Export Product category
  • Import / Export Product
  • Import / Export Product inventory
  • Import / Export Orders
  • Import / Export Customer Invoice
  • Import Customer payments
  • Import / Export Purchase order
  • Import Vendor Bills
  • Import Vendor bill payments
  • Import Customer Invoice

Generate The users credentials on Quickbooks Online for the Odoo, for generating the user credentials of quickbook online for the odoo click on the “OAuth 2.0 Playground” option in the users login on the Quickbook Online as seen in the image

As in the above image how to generate all the user credentials, copy all those required values (from right hand side menu also) which are required in odoo and paste them in odoo instance page as seen in the next section.

Instance Creation And Validation

In the instance form enter all the following details properly generated from quickbook online users credentials / playground page



After entering all the details, Authenticate the entered credentials by clicking the Authenticate

Now set the timezone of User (for refreshing / regenerating the access token) as seen below



Now install the required companies “Chart of Account” from Accounting menu as seen below

Regenerate/Refresh the Access Token, If the “Access Token” expires (as its validity is 60 minutes only) than you can simply click on the “Refresh Token” button from the instance form to regenerate it or we can run scheduler to automatically refresh the token.



Now perform the import functions, and from the “Manage Accounts” page of the instance form you can import the following as seen in below image
1. Accounts or Chart of Accounts
2. Taxes
3. Payment Methods
4. Payment term.

From the “Manage Customers And Vendors” tab import the following as seen in below snapshot
5. Departments
6. Customers
7. Vendors
8. Employees.

From the “Manage Products” tab on the instance import the following as seen in below snapshot
9. Product Category
10. Products (and Bundle Products also)
11. Products Inventory

From the “Manage Orders And Invoices” tab import the following as shown in below snapshot
12.Sales Order / Sales Receipt
13.Purchase Order
14.Invoices / Customer invoice
15. Bills / Vendor Bills,
16. Payments / Customer Payments
17. Bills Payments / Vendor Bills Payments.

We can import all these from the “Dashboard” menu also and from that properly select the instance by its provided / created name (as given on instance creation, for eg. “US” in our case as seen in below image),





We can set the cron to import all operations using Odoo scheduler.
